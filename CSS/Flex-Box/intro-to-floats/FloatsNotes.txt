##Responsive Design Defined by 3 characteristics
1- Flexible Grid Layout
2-Media Queries(CSS3)
3-Images that resize

/* rearranging the columns */
[] = attribute selector

[class*="col-"] {} : any html attribute with a class attributes that contain "col-
"
ex:
[href="http:"] {} - can tag all links that go offsite with any behavior you want

or

[http*=".PDF"] {} - you can stick a little PDF icon after every link that has a PDF.
